# BlockMiner

使用活塞回收特性将目标方块进行移除

开发中！

## 使用说明

#### 执行条件

使用前需准备好如下物品：

| 条件     |   数量/说明    |
|--------|:----------:|
| 活塞     |     2      |
| 拉杆     |     1      |
| 黏液块    |     可选     |
| 效率5钻石镐 | 推荐(增加破坏速度) |
| 急迫2信标  | 推荐(增加破坏速度) |

#### 开启方法1

**空手**右键支持的目标方块开启/关闭此模组（注意要**空手**）

#### 开启方法2

输入 `/blockMiner` 命令开启/关闭此模组

#### 启用后使用

选中受支持的方块，然后左键点击该方块
